# Minimalist Productivity Tools for Notion

這是一系列專為生產力打造的極簡小工具，特別優化為適合嵌入 **Notion** 或作為網頁側邊欄使用。本專案透過多款 AI 協作開發，追求極致的簡潔與實用性。

## 🤖 多 AI 協作開發
本專案由 **Miller** 主導，並根據功能需求與多款 AI 進行疊代開發：

- **版本一：標準番茄鐘 (Standard Pomodoro)**
  - **開發協力**：**Claude**
  - **特點**：大尺寸圓環，操作直觀，視覺平衡感佳。
- **版本二：內嵌按鈕番茄鐘 (Compact Pomodoro)**
  - **開發協力**：**Gemini**
  - **特點**：按鈕整合進圓環內部，極致縮小體積，適合 Notion 側邊欄。
- **版本三：隨手隨畫小白板 (Miller's Whiteboard)**
  - **開發協力**：**Claude**
  - **特點**：支援「物理級」即時擦除、獨立筆觸/橡皮擦大小、無限延伸畫布。

## 🚀 專案特點
- **極簡佈局**：移除所有冗餘文字與邊距，畫面專注於核心功能。
- **Notion 友善**：支援透過 GitHub Pages 嵌入，寬度與高度皆可完美適應 Notion。
- **即時反應**：優化座標校準，確保筆跡與鼠標同步，無延遲感。

## 🛠️ 工具使用說明

### 🍅 番茄鐘系列
1. **設定時間**：點擊中央數字即可輸入分鐘數。
2. **操作**：支援 Start / Pause / Reset。

### 🎨 隨手畫小白板
1. **畫筆 (✏️)**：可調整筆觸大小與三種核心配色。
2. **橡皮擦 (🧽)**：支援「經過即擦除」邏輯，並可獨立調整擦除半徑。
3. **拖移 (✋)**：可按住拖動，無限延伸你的創意空間。
4. **清除 (Clear)**：一鍵清空所有畫布內容。
