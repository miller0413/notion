# Minimalist Pomodoro Timers

這是一個專為生產力打造的極簡化番茄鐘工具，特別優化為適合嵌入 **Notion** 或作為側邊欄使用。

## 🤖 多 AI 協作開發
本專案由本人 Miller 與多款 AI 協作開發，展現不同模型的設計優勢：

- **版本一：標準極簡版**
  - **開發協力**：**Claude**
  - **設計重點**：大尺寸圓環與經典布局，視覺平衡感佳。
- **版本二：內嵌按鈕版**
  - **開發協力**：**Gemini**
  - **設計重點**：按鈕內嵌於圓環中，極致縮小體積，適合置頂與側邊欄。

## 🚀 專案特點
- **極簡設計**：移除所有多餘文字，直觀顯示倒數圓環。
- **置頂佈局**：介面緊湊，完全貼合瀏覽器頂部，不佔用空間。
- **Notion 友善**：支援透過 GitHub Pages 嵌入至 Notion Dashboard。
- **自定義時間**：點擊時鐘數字即可輸入分鐘數並重設。

## 🛠️ 使用說明
1. **設定時間**：點擊中央數字，輸入分鐘數後按 Enter。
2. **開始/暫停**：點擊 `Start` / `Pause` 切換狀態。
3. **重設**：點擊 `Reset` 回復初始時間。
